# A-Frame Starter Project

This is a minimal starter for building with plain HTML, CSS, JavaScript, and the A-Frame library.

## Structure

```text
aframe-vr-game/
├── index.html
├── src/
│   ├── scripts/
│   │   └── main.js
│   └── styles/
│       └── main.css
├── package.json
└── server.js
```

## Using Live Server

1. Open the project folder in VS Code.
2. Right-click `index.html` in the project root.
3. Choose **Open with Live Server**.

The root `index.html` is set up to load the CSS from `src/styles/main.css`, the JavaScript from `src/scripts/main.js`, and A-Frame from its CDN.

## Optional npm commands

- `npm start` serves the project root.

## What to edit

- Scene markup: `index.html`
- Styles: `src/styles/main.css`
- JavaScript behavior: `src/scripts/main.js`
